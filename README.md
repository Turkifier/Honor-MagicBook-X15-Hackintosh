#  Hackintosh: This Time On Honor MagicBook X15
An EFI file for the ones with i5-10210U, and no keyboard backlighting variant.
## What's Working?
*   Wi-Fi
*   BlueTooth
*   camera
*   trackpad
*   audio
*   brightness keys, volume keys
*   iServices
*   HDMI
*   power management
*   battery read-outs
*   DRM
*   RTC
*   NVRAM
*   sleep.
#  What's Not Working?
*   fingerprint sensor
*   secure boot.
